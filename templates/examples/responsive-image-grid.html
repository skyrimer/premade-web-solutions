Responsive image grid

An image grid is one of the unique ways to present pictures. Instead of a carousel or a slider, it allows for maintaining the vertical style and display of images without visually equally rows. This tutorial will show you the easiest way to do so without JavaScript.

Despite no visual rows, all pictures are wrapped in a big row with multiple columns with independent alignment inside. Therefore, you can put as many columns as you want. Yet make sure that the height of images will be roughly the same because big gaps in one of the columns will look bad. Here is my code example:

In CSS, you need to make the image grid a flex container. To make it adaptive, you must also throw flex-wrap: wrap. It allows defining the wrapping moment using the flex-basis property set to each column. Also, you need to set the max-width to the gallery column because the wrapped content will take the entire width of the available resulting in stretched images. Finally, it is common that you would like to style every image except the selected or hovered one. This can be done by applying a non-active style when the entire row is selected and cancelling it when the user hovers over the image. 


.image-gallery {
  display: flex;
  flex-wrap: wrap;
  align-items: center;
  justify-content: center;
  gap: 0.5rem;
  .gallery-column {
    flex: 1 1 400px;
    max-width: 50%;
    img {
      width: 100%;
      margin-top: 0.5rem;
      object-fit: cover;
      filter: none;
      transition: all 200ms ease-in-out;
    }
  }
  &:hover {
    .gallery-column img {
      filter: saturate(50%);
      &:hover {
        filter: saturate(140%);
      }
    }
  }
}

That's it. It is another short tutorial, yet helpful and quite powerful. If you want to visualise it, then follow the CodePen.
