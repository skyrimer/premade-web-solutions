Simple Sidebar

Sidebars as a primary navigation source have become more and more common on the modern web. The sidebar looks more aesthetic on a widescreen. It can usually be transformed into a small navigation panel at the bottom of the screen on the mobile, improving user accessibility relative to the old-fashioned hamburger. Despite its visual simplicity, some minor details must be considered. In this tutorial, I will explain how to build a sidebar. As it doesn't require any JavaScript, I will focus on CSS only and explain everything in detail.

In the HTML, there is nothing special. There are 3 sections: logo or top part, main navigation and the footer (it is commonly account or feedback links). Main navigation consists of links with icons and labels for it. It is vital to keep labels short, as it will be hard to maintain all the links on a mobile screen if it is pretty lengthy. In CSS, we have to declare 2 styles for the sidebar: desktop and mobile version. Let's start with the mobile version because it is more straightforward to explain. You just need to fix it to the bottom of the screen using position: fixed and use inset (shorthand from the top, right, bottom, left). Using an inset has one benefit: as the sidebar is positioned relative to the body, setting all zeros except on the side will ensure that all content won't go beyond the screen. Everything else is just simple CSS that is up to you. However, ensure that both aside and all the links have a display: flex. This will allow us to easily rotate content for the desktop version. In addition, there is no point in setting a fixed height, as there is nothing bond to it.

Switch to the desktop styling has to be done with the breakpoint. For the desktop, you have to change the inset to fix the sidebar on the left and rotate all the link content using flex-direction: row.

It might seem that the sidebar is done, but there are minor adjustments that we need to make. On the desktop, you need to add overflow: auto to make all the content accessible if the screen height is too small to fit all the content. You might also style the scrollbar to be smaller and transparent with the outline to fit the sidebar. In addition, you need to set the width to the desktop style. This is needed to add the same value to the margin-left of the page content wrapper. It will push the content to the left, leaving a place for the sidebar.


$sidebar-breakpoint: 45rem;
:root {
  --clr-accent: rgba(61, 245, 167, 1);
  --clr-main: #111827;
  --clr-secondary: #393e46;
  --clr-sidebar-link-hover: #374151;
  --clr-text: #ededed;
  --sidebar-width: 13rem;
  --container-width: 70rem;
  font-family: "Merriweather", serif;
}

aside {

  --section-margin: 1em;
  --link-gap: 0.75em;
  --link-fz: 1em;
  --link-fz-mobile: 2.2vw;
  position: fixed;
  inset: auto 0 0 0;
  display: flex;
  align-items: center;
  flex-direction: column;
  overflow: auto;
  background-color: var(--clr-main);
  color: var(--clr-text);
  @media screen and (min-width: $sidebar-breakpoint) {
    inset: 0 auto 0 0;
    width: var(--sidebar-width);
  }
  a {
    display: flex;
    gap: var(--link-gap);
    align-items: center;
    flex-direction: column;
    width: 100%;
    color: var(--clr-text);
    transition: all 0.1s ease-in-out;

      flex-direction: row;

    }
  }
  .sidebar-logo {
    display: none;
    @media screen and (min-width: $sidebar-breakpoint) {
      display: flex;
      margin-top: 2em;
      padding-inline: calc(var(--padding-x) * 2);
      svg {
        --size: 2em;
      }
      span {
        font-weight: bold;
      }
    }
  }
  .sidebar-navigation-divider {
    @media screen and (min-width: $sidebar-breakpoint) {
      width: 100%;
      border-top: 1px solid var(--clr-secondary);
    }
  }
  .sidebar-navigation-wrapper {
    width: 100%;
    padding-inline: var(--padding-x);
  }
  .sidebar-section {
    display: flex;
    align-items: center;
    width: 100%;
    margin-bottom: var(--section-margin);
    @media screen and (min-width: $sidebar-breakpoint) {
      flex-direction: column;

      border-radius: 0.25em;
      svg {
        stroke: currentColor;
      }
      span {
        font-size: min(var(--link-fz), var(--link-fz-mobile));
        @media screen and (min-width: $sidebar-breakpoint) {
          font-size: var(--link-fz);
        }
      }
      &[aria-current="page"],

  }

      display: flex;
      margin-top: auto;
      justify-content: center;

  }
}

  }

  margin: 0 auto;
}
