Fullscreen hero section

Full-screen hero sections with a background image or video became a standard welcome screen on many websites. It is not complicated, but there are a few tricks and tips that I can show that will make your life easier.

In HTML, you need to make a container with your background element (video or image - it does not matter) and the text wrapper. Note that if you want to have a video playing, you have to put autoplay muted and loop attributes for everything to work accordingly. In addition, having a container for the elements above allows you to position the text elements in any way you want and align them with the width of other content on the page. Here is my example: 


<div class="hero-section" id="footer">
  <video class="hero-background" src="https://joy.videvo.net/videvo_files/video/free/2019-03/large_watermarked/190111_08_BuildingsTraffic_Drone_09_preview.mp4" autoplay muted loop alt="city"></video>
  <div class="hero-text container">
    <h2>Goodbye to TOKIO</h2>
    <a class="button" href="#header">
      <span class="text">Go to top hero</span>
    </a>
  </div>
</div>

In CSS, you need to focus on a few things. Setting min-height: 100vh instead of height benefits small screens: no overflow if the text takes up more than the entire screen. Remember to set the position: relative to the container: this will be needed for positioning background and background effects. Talking about them, if you want to have some filters like darkening or hue-rotation, you can add the extra layer of a pseudo-element between the text and the background by messing around with the z-index or using isolation: isolate. Finally, it is vital to stretch the background using width and height properties and to set top: 0 and left: 0. This way, you can apply the object-fit property, and it will work as expected.

Here is a small tip: if you want to have a similar effect to background-attachment: fixed, you can set clip-path: inset(0) to the container and position: sticky to your background:


.hero-section {
  position: relative;
  width: 100%;
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: start;
  clip-path: inset(0);
  &::before {
    content: "";
    position: absolute;
    inset: 0;
    z-index: -1;
    background-image: linear-gradient(
      to right,
      rgba(0, 0, 0, 0.8),
      rgba(0, 0, 0, 0.5),
      transparent
    );
  }
  .hero-background {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    object-fit: cover;
    z-index: -2;
  }
  .hero-text {
    z-index: 2;
    h2 {
      color: white;
      font-size: 6rem;
      margin-bottom: 0.5em;
    }
  }
}
