Floating labels for inputs

One of the best thing you can do while creating forms is to label every single input with its label. It make the website more intuitive and user-friendly. One of the ways to add some fancy micro-interactions for the website is to make the labels float.

Floating label is a term that is used to demonstrate that the label moves as soon as user focuses the input. You can see it on the example below:

In terms of HTML, structure maters a lot. You have to have a wrapper with an input followed by a label. It is important to have a label afterwards because we are using pure CSS. For this example, you will also need a placeholder for the input. There are other ways to do it using :focus and :valid, but we find this one a bit simpler. 


<div class="input-wrapper">
  <input type="text" id="input" placeholder="Put any text here">
  <label for="input">Click</label>
</div>

For CSS, you can style the input and label however you want, however, there are some things to think of. You have to set position: relative to the wrapper, so you can position the label. Another thing to be watched is padding: you have to make sure that there is enough space both for the input text and for the label text. Here, we are lowering the input text by setting greater padding-top than padding-bottom so there is enough space for label text.

Finally, we are using the magic selector input:placeholder-shown + label to change styling for the label. This selector will cover all the labels after input when the placeholder value is not visible. In particular, we are moving the label higher using transform: translate(), but you can also use top: and left: 


input {
  font-size: 1.2rem;
  width: min(100%, 50rem);
  padding: 2em 2rem 0.5em;
  color: var(--color-text);
  background-color: var(--color-background);
  border-bottom: 1px solid var(--color-background);
  border-radius: 0.5em;
}

/* So the placeholder does not overlap with the label */
input::placeholder {
  color: transparent;
}

input:placeholder-shown+label {
  transform: translateY(-50%);
  font-size: 1rem;
}

label {
  font-size: 0.9rem;
  transform: translate(0, calc(-50% - 1em));
  user-select: none;
  position: absolute;
  top: 50%;
  left: 2rem;
  color: var(--color-accent);
  transition: all 0.3s ease-in-out;
}
